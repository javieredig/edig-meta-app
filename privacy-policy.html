Política de Privacidad

Ediciones Digitales y Desarrollo de Software
Última actualización: junio 2026

1. ¿Quiénes somos?

Ediciones Digitales y Desarrollo de Software (en adelante "Edig") es una empresa de tecnología. Esta aplicación conecta nuestras herramientas internas con la API de Meta con el fin de gestionar y analizar campañas publicitarias de manera automatizada.

2. ¿Qué datos recopilamos?

  • Datos de campañas publicitarias de Meta Ads (nombres, métricas, presupuestos, audiencias).
  • Tokens de acceso a la API de Meta, utilizados exclusivamente para autenticación.
  • No recopilamos datos personales de usuarios finales ni de terceros.

3. ¿Cómo usamos los datos?

Los datos obtenidos a través de la API de Meta se utilizan únicamente para:

  • Analizar el rendimiento de campañas publicitarias internas de Edig.
  • Crear y modificar conjuntos de anuncios (ad sets) de forma automatizada.
  • Generar reportes internos para el equipo de marketing.

Los datos no se comparten con terceros ni se utilizan con fines comerciales externos.

4. ¿Dónde se almacenan los datos?

Los datos son procesados en servidores internos de Edig. No se almacena información de Meta de forma permanente más allá de lo necesario para el funcionamiento del sistema.

5. Seguridad

Implementamos medidas de seguridad razonables para proteger los tokens de acceso y los datos de campañas frente a accesos no autorizados.

6. Cambios a esta política

Podemos actualizar esta política ocasionalmente. Los cambios serán publicados en esta misma página.
